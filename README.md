# l4d2_coopmode
求生之路2的战役模式包，服务器端安装完毕，将本项目克隆或者下载，解压到对应目录即可游玩

#### 游戏模式包含

- **单人模式**
  - 有大枪
  - 无大枪
  - hunter训练
- **多人模式**
  - 药役模式（基于战役模式修改）
  - hunter派对（全是hunter的模式）
  - 牛牛快跑（全是charger的模式）
  - 坦克润（基于tank run模式修改，增加坦克刷新频率，狙击类武器攻击倒地队友起身，小手枪射出舌头飞檐走壁，马格南卫星炮等）
  - 纯净模式（还原官方模式原汁原味，只有部分提示等插件）
  - 浓雾模式（能见度极低的一种模式）
- **生还者对抗生还者**
  - 固定回合模式
    - 固定回合模式，开局倒数时间内玩家无敌，下蹲可加速移动
    - 60秒内无人死亡，开启全局透视，所有玩家发光几秒，加速回合，防止玩家只蹲不进攻
    - 等待死亡人数为1时复活死亡玩家，重新开始回合
    - 每关3回合结束，传送安全屋开始下一关
    - 救援关5回合结束，不再复活玩家，可以选择自杀重开，或者坐载具离开，投票换图等
  - 随机重生模式
    - 随机复活模式，开局倒数时间内玩家无敌
    - 回合开始弹出菜单可选技能，shift + e 使用技能
    - 死亡玩家倒计时后随机位置复活
    - 每关累计击杀数量到达上限后，传送安全屋开始下一关
    - 救援关没有上限，想要结束请自主换图

#### 游戏投票包含

- 部分突变模式（有的突变模式和插件互相排斥，所以大部分都处于锁定状态，不能切换）
- 地图版本切换（zonemod、neomod、deadmod）
- 特感数量，刷新时间，刷特模式修改
- HUD修改
- 商店相关设置（包含开关商店，医疗物品购买，传送）
- 队友伤害相关（默认开启队友伤害惩罚，队伤到达一定程度会倒地甚至处死）
- 坦克相关（连跳，关卡单克）
- 击杀特感回血回子弹（想要修改回血量或者子弹量，可以去cfg/vt_cfg寻找相关代码；或者修改插件源码重新编译）
- 过关复活，死亡复活
- 武器伤害（求生默认伤害，zone，anne，anne+伤害，以及开关大枪）
- 枪械抖动开关
- 右键推数设置
- 多倍医疗设置
- 删除地图医疗资源
- 单人模式增加高级AI特感投票
- 纯净模式增加连跳投票开关
